API Documentation

1. GET /tasks

Description: Fetches the current task data stored in the database.

Request

Method: GET
URL: /tasks

Response

Status Code: 200 OK

{
  "pendingTasks": 55,
  "inProgressTasks": 30,
  "completeTasks": 20,
  "cancelTasks": 10,
  "partialTasks": 5,
  "speedUpTasks": 7,
  "totalAmount": 1345
}

Status Code: 404 Not Found (If no task data exists)

{
  "message": "No task data found."
}

Status Code: 500 Internal Server Error (If there's an error fetching the data)

{
  "message": "Error fetching task data."
}

2. PUT /tasks/update

Description: Updates the task values in the database. If no data exists, it will create a new entry with the provided values.

Request

Method: PUT
URL: /tasks/update

Body

{
  "pendingTasks": 55,
  "inProgressTasks": 30,
  "completeTasks": 20,
  "cancelTasks": 10,
  "partialTasks": 5,
  "speedUpTasks": 7,
  "totalAmount": 1345
}

Response

Status Code: 200 OK

{
  "message": "Task data updated successfully.",
  "taskData": {
    "pendingTasks": 55,
    "inProgressTasks": 30,
    "completeTasks": 20,
    "cancelTasks": 10,
    "partialTasks": 5,
    "speedUpTasks": 7,
    "totalAmount": 1345
  }
}

Status Code: 500 Internal Server Error (If there's an error updating the data)

{
  "message": "Error updating task data."
}

Server API URL: http://159.223.156.179:5000/
